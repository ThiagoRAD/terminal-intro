## Some commands

### Print something to the terminal: echo
```
echo Hello!
```

### Print working directory: pwd
```
pwd
```

### Change directory: cd
```
cd directory-name
```

### List content: ls
```
ls
```
### Traversing the file system
```
pwd
/Users/youruser/somefolder
cd ..
/Users/youruser
```
* using the ```..``` double period moves us back in our filesystem
```
pwd
/Users/youruser/somefolder
cd ../..
/Users
```
